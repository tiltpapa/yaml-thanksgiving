# AGENTS.md

このリポジトリで作業するAIエージェント向け運用ガイド。

## 適用方針
- 常時適用

## 言語設定
- 回答は常に日本語
- 体言止めを意識
- 自明な助詞（が/の/を/に/で/は/と/も）は意味通じる範囲で省略
- 聞かれたことだけ回答

## プロジェクト前提
- 旧Reveal.jsプロジェクト `https://github.com/tiltpapa/reveal-thanksgiving` の改善
- YAML編集中心でクイズスライドを生成できる体験へ改善

## プロジェクト目標
- 択一クイズ表示スライド
- 択数可変・画像対応テンプレート
- 正答表示対応
- 外部API連携による回答数表示・ランキング表示対応

## タスク整理
- タスク整理は `task.md` で実施

## コーディング規定
- CSS
  - `!important` 不使用
  - 強いセレクター利用は極力回避
  - ユーザー編集済み細かなパラメータ再編集時は事前確認
- 機能分離
  - UIロジックとデータ取得ロジックを分離し、ファイル分割
  - 各モジュール単一責任
  - 依存関係は一方向（上位→下位）
  - HTML内JavaScript記述は最小限

## MCP（外部知識参照）
- CSSパラメータ調整時は `chrome-devtools` 参照
  - アクセス時キャッシュ無効化
- 旧プロジェクト参照時は `reveal-thanksgiving` 利用
